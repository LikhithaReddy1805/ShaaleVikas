# Shaale Vikas – School Alumni Bridge Application

## Overview
Shaale Vikas is an Android application developed to improve transparency and collaboration between school administrators and alumni contributors for school development activities. The application provides a centralized platform where schools can post infrastructure and development needs, while alumni users can contribute support and monitor the progress of initiatives in real time.

The project was developed using Kotlin, Jetpack Compose, Firebase Authentication, and Firebase Firestore as part of an industry-oriented internship program focused on modern Android application development and cloud backend integration.

---

# Problem Statement
Many schools face challenges in managing infrastructure development and fundraising activities due to lack of transparency, limited alumni engagement, and inefficient communication systems. Traditional manual processes make it difficult to track contributions, monitor progress, and maintain proper coordination between schools and supporters.

Shaale Vikas addresses this problem by providing a digital platform that connects administrators and alumni through transparent contribution tracking, real-time updates, and centralized school development management.

# Installation Steps

## 1. Clone the Repository

```bash
git clone https://github.com/LikhithaReddy1805/ShaaleVikas.git
```

---

## 2. Open in Android Studio

Open Android Studio and select:

```text
Open Existing Project
```

Choose the cloned repository folder.

---

## 3. Sync Gradle

Allow Gradle files and dependencies to sync completely.

---

## 4. Configure Firebase

- Create a Firebase project
- Enable Firebase Authentication
- Enable Firebase Firestore
- Download `google-services.json`
- Place it inside:

```text
app/google-services.json
```

---

## 5. Run the Application

Connect:
- Android Emulator
OR
- Physical Android Device

Then click:

```text
Run ▶
```
