# Shaale Vikas – School Alumni Bridge Application

## Overview
Shaale Vikas is an Android application developed to improve transparency and collaboration between school administrators and alumni contributors for school development activities.

## Features
- Role-based Login System
- Firebase Authentication
- Firebase Firestore Integration
- Contribution Tracking
- Hall of Fame
- Impact Stories
- Responsive UI using Jetpack Compose
- Dark Mode Support

## Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Material Design

## Project Structure
- UI Screens
- Firebase Integration
- Navigation System
- Authentication Module
- Admin and Alumni Modules
